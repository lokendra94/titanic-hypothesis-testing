# Titanic Survival Hypothesis Testing

This project analyzes the Titanic dataset using Python to find which factors affect passenger survival.

## Hypothesis Tests Performed
- Chi-square: Sex vs Survival
- T-test: Age difference in survivors vs non-survivors
- ANOVA: Survival by passenger class
- Correlation: Age vs Fare

## Files Included
- titanic_hypothesis_project.py
- titanic_report.pdf
- Dataset (from seaborn)
- Visualizations & results

## Libraries Used
- pandas
- seaborn
- scipy.stats
